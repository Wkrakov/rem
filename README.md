# 📚 TimeStudent - Персональный планировщик задач

## 📋 Описание
TimeStudent - это приложение для управления задачами с функциями приоритетов, прогресса выполнения, статистики, анимаций и календаря. Каждый пользователь имеет свои персональные данные.

## 🚀 Возможности
- 🔐 Персональная аутентификация пользователей
- 📝 Создание и управление задачами
- 🎯 Приоритеты задач с цветовой кодировкой
- 📊 Прогресс выполнения с визуализацией
- 📈 Статистика по задачам
- 🎉 Анимации и поздравления при достижении 100%
- 📅 Календарь на русском языке
- 📂 Разделение данных между пользователями
